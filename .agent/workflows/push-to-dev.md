# Push to dev workflow

This workflow ensures that all changes are staged, committed, and pushed to the `dev` branch as per the user's rule.

// turbo-all
1. Navigate to the project root
2. Stage all changes: `git add .`
3. Commit changes: `git commit -m "[Descriptive Message]"`
4. Push to dev: `git push origin dev`
